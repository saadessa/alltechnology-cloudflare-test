# تثبيت Claude Code: دليل التثبيت على Windows و macOS و Linux 2026

قبل أن تبدأ في استخدام Claude Code، تحتاج إلى تثبيته على جهازك. هذه العملية بسيطة وتستغرق بضع دقائق فقط.

هذا الدليل يغطي جميع طرق تثبيت Claude Code على أنظمة التشغيل المختلفة: Windows و macOS و Linux.

للتعرف على Claude Code بشكل عام، يمكنك قراءة المقالة الرئيسية [Claude Code: الدليل الشامل للمبتدئين والمحترفين في 2026](/posts/claude-code-guide-2026).

---

## متطلبات التشغيل الأساسية

قبل تثبيت Claude Code، تأكد من توفر هذه المتطلبات:

### البرامج المطلوبة

1. **Node.js:** الإصدار 18 أو أحدث
   - حمله من [nodejs.org](https://nodejs.org)
   - تأكد من التثبيت بـ `node --version`

2. **Git:** الإصدار 2.38 أو أحدث
   - حمله من [git-scm.com](https://git-scm.com)
   - تأكد من التثبيت بـ `git --version`

3. **اتصال بالإنترنت:** للتواصل مع خوادم Anthropic

### الحساب المطلوب

- **حساب Claude:** اشتراك Pro أو Max
- **طريقة الدخول:** يمكن استخدام حساب Claude، API Key، Amazon Bedrock، أو Google Vertex AI

---

## طرق تثبيت Claude Code

### الطريقة الأولى: عبر npm (الموصى بها لجميع الأنظمة)

هذه هي الطريقة الأكثر شيوعاً وتعمل على جميع أنظمة التشغيل.

**الخطوات:**

1. افتح الطرفية (Terminal):
   - **Windows:** Command Prompt أو PowerShell
   - **macOS:** Terminal
   - **Linux:** Terminal

2. شغل الأمر التالي:

```bash
npm install -g @anthropic-ai/claude-code
```

3. انتظر حتى يكتمل التثبيت (قد يستغرق دقيقة أو دقيقتين)

4. تحقق من التثبيت:

```bash
claude --version
```

> **⚠️ تنبيه مهم:** لا تستخدم `sudo` مع npm مهما كان النظام. هذا يسبب مشاكل في صلاحيات الملفات لاحقاً.

---

### الطريقة الثانية: عبر Homebrew (macOS)

إذا كنت تستخدم macOS وتفضل Homebrew، يمكنك التثبيت بهذه الطريقة.

**الخطوات:**

1. تأكد من تثبيت Homebrew على جهازك:

```bash
brew --version
```

2. شغل الأمر التالي:

```bash
brew install --cask claude-code
```

3. تحقق من التثبيت:

```bash
claude --version
```

---

### الطريقة الثالثة: عبر WinGet (Windows)

لمستخدمي Windows الذين يفضلون WinGet.

**الخطوات:**

1. افتح PowerShell

2. شغل الأمر التالي:

```bash
winget install Anthropic.ClaudeCode
```

3. تحقق من التثبيت:

```bash
claude --version
```

---

### الطريقة الرابعة: عبر المثبت الرسومي (macOS و Windows)

إذا كنت تفضل واجهة رسومية بدلاً من سطر الأوامر.

**الخطوات:**

1. اذهب إلى [claude.com/download](https://claude.com/download)

2. اختر نظام التشغيل الخاص بك:
   - **macOS:** حمل ملف `.dmg`
   - **Windows:** حمل ملف `.exe`

3. افتح المثبت واتبع التعليمات

4. بعد التثبيت، افتح الطرفية وتحقق:

```bash
claude --version
```

## تثبيت Claude Code على أنظمة التشغيل المختلفة

### تثبيت Claude Code على Windows

**الخيار 1: عبر npm (موصى به)**

1. ثبّت Node.js من [nodejs.org](https://nodejs.org)
2. افتح Command Prompt كمسؤول
3. شغل: `npm install -g @anthropic-ai/claude-code`
4. تحقق: `claude --version`

**الخيار 2: عبر WinGet**

1. افتح PowerShell
2. شغل: `winget install Anthropic.ClaudeCode`
3. تحقق: `claude --version`

**الخيار 3: عبر المثبت الرسومي**

1. حمل من [claude.com/download](https://claude.com/download)
2. شغل المثبت واتبع التعليمات

---

### تثبيت Claude Code على macOS

**الخيار 1: عبر npm (موصى به)**

1. ثبّت Node.js من [nodejs.org](https://nodejs.org)
2. افتح Terminal
3. شغل: `npm install -g @anthropic-ai/claude-code`
4. تحقق: `claude --version`

**الخيار 2: عبر Homebrew**

1. افتح Terminal
2. شغل: `brew install --cask claude-code`
3. تحقق: `claude --version`

**الخيار 3: عبر المثبت الرسومي**

1. حمل من [claude.com/download](https://claude.com/download)
2. افتح ملف `.dmg` واسحب Claude Code إلى مجلد التطبيقات

---

### تثبيت Claude Code على Linux

**الخيار 1: عبر npm (الطريقة الوحيدة)**

1. ثبّت Node.js من [nodejs.org](https://nodejs.org) أو عبر مدير الحزم:
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install nodejs npm
   
   # Fedora
   sudo dnf install nodejs
   ```

2. ثبّت Git:
   ```bash
   # Ubuntu/Debian
   sudo apt install git
   
   # Fedora
   sudo dnf install git
   ```

3. شغل npm:
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

4. تحقق: `claude --version`

> **⚠️ تنبيه:** لا تستخدم `sudo` مع npm. إذا ظهرت مشكلة صلاحيات، استخدم مدير الحزم لتصحيحها.

## مشاكل التثبيت الشائعة وحلولها

### مشكلة: "command not found: claude"

**السبب:** لم يتم تثبيت Claude Code بشكل صحيح، أو أن المسار غير مضاف إلى PATH.

**الحل:**

1. تأكد من التثبيت بـ `npm install -g @anthropic-ai/claude-code`
2. أعد تشغيل الطرفية
3. إذا استمرت المشكلة، أعد تثبيت Node.js

### مشكلة: "npm: command not found"

**السبب:** Node.js غير مثبت على جهازك.

**الحل:** ثبّت Node.js من [nodejs.org](https://nodejs.org)

### مشكلة: "EACCES: permission denied"

**السبب:** مشكلة صلاحيات عند استخدام npm.

**الحل:** لا تستخدم `sudo` أبداً. بدلاً من ذلك:
- على macOS/Linux: استخدم مدير الحزم أو ثبّت Node.js عبر nvm
- على Windows: شغل Command Prompt كمسؤول

### مشكلة: "brew command not found" (macOS)

**السبب:** Homebrew غير مثبت.

**الحل:** ثبّت Homebrew من [brew.sh](https://brew.sh)

### مشكلة: "winget command not found" (Windows)

**السبب:** WinGet غير مثبت أو غير محدث.

**الحل:** استخدم npm بدلاً من ذلك، أو حدث WinGet عبر Microsoft Store.
