## ما هو Claude Code ولماذا هو مهم للمبتدئين؟

Claude Code هو أداة برمجة وكيلة (Agentic Coding Tool) من Anthropic. بعبارة بسيطة: هو مساعد برمجي ذكي يفهم مشروعك بالكامل ويساعدك في كتابة وتعديل وإصلاح الكود.

**لماذا هو مثالي للمبتدئين؟**

- **لا تحتاج حفظ أوامر معقدة:** تتحدث معه بلغة طبيعية
- **يفهم السياق الكامل:** يقرأ مشروعك بالكامل وليس فقط السطر الذي تعمل عليه
- **يساعدك في التعلم:** يشرح الكود بلغة بسيطة
- **يكتب الكود لك:** يمكنك طلب كتابة دوال أو إصلاح أخطاء بسهولة

ببساطة، Claude Code هو بمثابة خبير برمجي بجانبك يساعدك في كل خطوة.

## خطوات تثبيت Claude Code للمبتدئين

### الخطوة الأولى: تثبيت Claude Code

افتح الطرفية (Terminal) على جهازك:

**لـ Windows:** ابحث عن "Command Prompt" أو "PowerShell"

**لـ macOS:** ابحث عن "Terminal"

**لـ Linux:** ابحث عن "Terminal"

ثم اكتب الأمر التالي:

```bash
npm install -g @anthropic-ai/claude-code
```

> **ملاحظة للمبتدئين:** إذا طلب منك استخدام `sudo`، لا تفعل. هذا يسبب مشاكل في الصلاحيات لاحقاً.

### الخطوة الثانية: التحقق من التثبيت

```bash
claude --version
```

إذا ظهر رقم الإصدار، التثبيت تم بنجاح.

### الخطوة الثالثة: تسجيل الدخول

```bash
claude login
```

اتبع التعليمات في الطرفية والمتصفح لتسجيل الدخول بحساب Claude الخاص بك.

للحصول على دليل مفصل لجميع طرق التثبيت، راجع [تثبيت Claude Code: دليل التثبيت على Windows و macOS و Linux](/posts/claude-code-installation-guide).


ستظهر رسالة ترحيب وجاهزية لاستقبال أوامرك.

### الخطوة الثالثة: اكتب أول أمر

> "اكتب دالة بلغة JavaScript تقوم بجمع عددين وإرجاع النتيجة."

سيرى Claude Code أنك في مشروع جديد، وسيكتب لك دالة بسيطة، وسيشرح لك كيف تعمل.

### الخطوة الرابعة: تفاعل مع الردود

بعد كتابة الأمر، سيُظهر Claude Code الكود المقترح. يمكنك:

- **قبول التغيير:** بالموافقة على إنشاء الملف
- **رفض التغيير:** إذا لم يعجبك الحل
- **طلب تعديل:** "عدل الدالة لتقبل 3 أرقام بدلاً من 2"

## الأوامر الأساسية للمبتدئين

### 1. طلب شرح كود

إذا كان لديك كود لا تفهمه، فقط الصقه واطلب شرحاً.

> "اشرح لي هذا الكود بلغة بسيطة: [الصق الكود]"

### 2. طلب كتابة كود

> "اكتب دالة بلغة Python تقوم بحساب متوسط قائمة أرقام"

### 3. طلب إصلاح خطأ

إذا ظهر لك خطأ، انسخ رسالة الخطأ والكود:

> "لدي هذا الخطأ: [الخطأ] في هذا الكود: [الكود]. كيف أصلحه؟"

### 4. طلب تحسين كود

> "هذا الكود يعمل لكنه بطيء. كيف أحسّن أداءه؟ [الكود]"

### 5. طلب إنشاء ملف كامل

> "أنشئ ملف index.html يحتوي على صفحة بسيطة بعنوان 'مرحباً بالعالم'"

للحصول على قائمة كاملة بالأوامر واستخداماتها، راجع [أوامر Claude Code الأساسية: دليل سريع للمبرمجين](/posts/claude-code-commands-guide).

## نصائح مهمة للمبتدئين

### 1. ابدأ بمشاريع صغيرة

لا تبدأ بمشروع كبير معقد. جرب Claude Code أولاً على مشروع صغير لفهم كيف يعمل.

### 2. استخدم اللغة الطبيعية

لا تحتاج إلى صياغة أوامر تقنية معقدة. تحدث مع Claude Code كما تتحدث مع زميل.

**أمر جيد:** "ساعدني في كتابة دالة تقوم بحساب مساحة المستطيل"

**أمر سيء:** "code function rectangle area"

### 3. اطلب الشرح

لا تكتفِ بالكود، اطلب شرحاً. هذا يساعدك في التعلم.

> "اشرح لي لماذا اخترت هذا الحل بدلاً من حل آخر."

### 4. استخدم المحادثات الطويلة

استمر في نفس المحادثة بدلاً من بدء محادثة جديدة لكل سؤال. هذا يحافظ على السياق ويساعد Claude Code على فهم مشروعك بشكل أفضل.

### 5. ثق ولكن تحقق

Claude Code أداة قوية لكنها ليست معصومة. تحقق من الكود قبل استخدامه في مشروعك الحقيقي.
