# أوامر ChatGPT لوسائل التواصل الاجتماعي: دليل صناع المحتوى 2026

في عام 2026، أصبحت وسائل التواصل الاجتماعي جزءاً لا يتجزأ من استراتيجيات التسويق وبناء العلامات التجارية. لكن إنتاج محتوى مستمر وجذاب لمنصات متعددة يمكن أن يكون مرهقاً ويستهلك وقتاً طويلاً. هنا يأتي دور ChatGPT كمساعد قوي يمكنه توليد أفكار، كتابة نصوص، واقتراح استراتيجيات تناسب كل منصة.

هذا الدليل يقدم لك مجموعة من الأوامر الفعالة (Prompts) المصممة خصيصاً لوسائل التواصل الاجتماعي، مع أمثلة عملية لكل منصة، ونصائح للحصول على أفضل النتائج.

---

## 1. أساسيات كتابة أوامر وسائل التواصل الاجتماعي

### 1.1 هيكل الأمر المثالي

الأمر الفعّال لمحتوى وسائل التواصل يجب أن يحتوي على:

1. **المنصة**: إنستغرام، تويتر، لينكد إن، فيسبوك، تيك توك.
2. **نوع المحتوى**: منشور، قصة، فيديو، خيط تغريدات، مقال.
3. **الموضوع**: ما هو الموضوع الرئيسي؟
4. **الجمهور**: من تستهدف؟ (رواد أعمال، طلاب، محترفون، إلخ)
5. **النغمة**: مرحة، ملهمة، تعليمية، مهنية، جريئة.
6. **الهدف**: زيادة التفاعل، بناء مجتمع، توعية، مبيعات.
7. **الطول**: قصير، متوسط، طويل.
8. **عناصر إضافية**: هاشتاغات، دعوة للتفاعل (CTA)، روابط.

### 1.2 قالب الأمر الأساسي

```
اكتب [نوع المحتوى] لـ [المنصة] عن [الموضوع]، يستهدف [الجمهور].
النغمة: [وصف النغمة]
الهدف: [وصف الهدف]
الطول: [قصير/متوسط/طويل]
المطلوب: [وصف المخرجات المحددة]
```

## 2. أوامر إنستغرام (Instagram)

### 2.1 كتابة نصوص المنشورات (Captions)

**الأمر**:
```
اكتب [عدد] نصوصاً لمنشورات إنستغرام عن [الموضوع]، تستهدف [الجمهور].
النغمة: [مرحة، ملهمة، تعليمية]
الهدف: [زيادة التفاعل، التوعية، المبيعات]
الطول: [قصير/متوسط]
تشمل: دعوة للتفاعل (CTA) واقتراح هاشتاغات.
```

**مثال عملي**:
```
اكتب 3 نصوصاً لمنشورات إنستغرام عن نصائح العناية بالبشرة، تستهدف النساء في العشرينات والثلاثينات.
النغمة: ملهمة وودودة.
الهدف: زيادة التفاعل وبناء مجتمع.
الطول: متوسط (150-200 كلمة).
تشمل: دعوة للتفاعل (شاركنا روتينك المفضل) واقتراح 10 هاشتاغات مناسبة.
```

### 2.2 توليد أفكار لـ Reels

**الأمر**:
```
اقترح [عدد] أفكاراً لفيديوهات Reels على إنستغرام عن [الموضوع]، مع وصف لكل فكرة، والموسيقى المقترحة، والهاشتاغات المناسبة.
```

**مثال عملي**:
```
اقترح 5 أفكار لفيديوهات Reels عن وصفات صحية سريعة، مع وصف لكل فكرة (مدة 15-30 ثانية)، والموسيقى المقترحة (نوع الأغنية)، والهاشتاغات المناسبة.
```

### 2.3 كتابة نصوص القصص (Stories)

**الأمر**:
```
اكتب [عدد] نصوصاً لقصص إنستغرام (Stories) عن [الموضوع]، مع اقتراح عناصر تفاعلية (استطلاع، سؤال، شريط تمرير).
```

**مثال عملي**:
```
اكتب 5 نصوصاً لقصص إنستغرام عن إطلاق منتج جديد، مع اقتراح عناصر تفاعلية (استطلاع رأي عن اللون المفضل، سؤال عن التوقعات، شريط تمرير لتقييم الحماس).
النغمة: مثيرة وحماسية.
الهدف: بناء الترقب وجذب الانتباه.
```

### 2.4 استراتيجية محتوى شهرية

**الأمر**:
```
ضع خطة محتوى شهرية لإنستغرام لـ [العلامة التجارية/الموضوع]، تشمل: 20 فكرة منشور، 10 أفكار لـ Reels، 10 أفكار لـ Stories، مع توضيح الموضوعات الأسبوعية ونوع المحتوى لكل يوم.
```

**مثال عملي**:
```
ضع خطة محتوى شهرية لعلامة تجارية في مجال اللياقة البدنية، تشمل: 20 فكرة منشور (تمارين، نصائح تغذية، تحفيز)، 10 أفكار لـ Reels (تمارين سريعة، تحديات)، 10 أفكار لـ Stories (أسئلة وأجوبة، استطلاعات)، مع توضيح الموضوعات الأسبوعية (الأسبوع الأول: تمارين، الثاني: تغذية، الثالث: تحفيز، الرابع: قصص نجاح).
```

### 2.5 الردود على التعليقات

**الأمر**:
```
اقترح [عدد] ردوداً على تعليقات المتابعين على منشور عن [الموضوع]، مع الحفاظ على نبرة ودودة ومهنية.
```

**مثال عملي**:
```
اقترح 5 ردوداً على تعليقات المتابعين على منشور عن نصائح العناية بالبشرة، مع الحفاظ على نبرة ودودة ومهنية.
التعليقات تشمل: شكر، سؤال عن منتج معين، مشاركة تجربة شخصية، طلب نصائح إضافية.
```

## 3. أوامر تويتر (Twitter / X)

### 3.1 كتابة تغريدات جذابة

**الأمر**:
```
اكتب [عدد] تغريدات عن [الموضوع]، تستهدف [الجمهور].
النغمة: [ذكية، جريئة، إعلامية، فكاهية]
الهدف: [زيادة التفاعل، جذب المتابعين، نشر الوعي]
الطول: قصير (تحت 280 حرفاً)
تشمل: هاشتاغ واحد أو اثنين.
```

**مثال عملي**:
```
اكتب 5 تغريدات عن أهمية الذكاء الاصطناعي في التسويق الرقمي، تستهدف المسوقين وأصحاب الأعمال.
النغمة: ذكية ومثيرة للتفكير.
الهدف: جذب المتابعين وإظهار الخبرة.
الطول: قصير (تحت 280 حرفاً).
تشمل: هاشتاغ واحد مناسب (#AI #DigitalMarketing).
```

### 3.2 كتابة خيوط تغريدات (Threads)

**الأمر**:
```
اكتب خيط تغريدات (Thread) عن [الموضوع] يتكون من [عدد] تغريدات، بحيث تكون كل تغريدة مكملة للأخرى وتنتهي بدعوة للتفاعل.
```

**مثال عملي**:
```
اكتب خيط تغريدات عن خطوات بناء استراتيجية تسويق ناجحة يتكون من 7 تغريدات، بحيث تكون كل تغريدة مكملة للأخرى وتنتهي بدعوة للتفاعل (شاركنا تجربتك، أعد تغريد الخيط).
النغمة: تعليمية وعملية.
```

### 3.3 كتابة ردود على التغريدات الشائعة

**الأمر**:
```
اكتب [عدد] ردوداً على التغريدة التالية، مع الحفاظ على نبرة تتناسب مع المحتوى الأصلي:
[نص التغريدة]
```

**مثال عملي**:
```
اكتب 3 ردوداً على التغريدة التالية، مع الحفاظ على نبرة إيجابية ومفيدة:
"ما هي أفضل أداة ذكاء اصطناعي لإنشاء المحتوى في 2026؟"
الردود: رد واحد مفيد بتوصية أداة، رد واحد يطلب توضيحاً عن نوع المحتوى، رد واحد يشارك تجربة شخصية.
```

### 3.4 توليد استطلاعات رأي (Polls)

**الأمر**:
```
أنشئ استطلاع رأي على تويتر عن [الموضوع] مع [عدد] خيارات، مع نص جذاب للاستطلاع.
```

**مثال عملي**:
```
أنشئ استطلاع رأي على تويتر عن أفضل منصة للتواصل الاجتماعي للتسويق مع 4 خيارات (إنستغرام، تويتر، لينكد إن، تيك توك)، مع نص جذاب للاستطلاع يدعو للمشاركة.
```

## 4. أوامر لينكد إن (LinkedIn)

### 4.1 كتابة منشورات احترافية

**الأمر**:
```
اكتب منشوراً لـ LinkedIn عن [الموضوع]، يستهدف [الجمهور المهني].
النغمة: مهنية ومقنعة
الهدف: [إظهار الخبرة، جذب العملاء، بناء شبكة علاقات]
الطول: متوسط إلى طويل (300-500 كلمة)
يتضمن: مقدمة جذابة، نقاط رئيسية، وخاتمة مع دعوة للتفاعل.
```

**مثال عملي**:
```
اكتب منشوراً لـ LinkedIn عن مستقبل الذكاء الاصطناعي في مجال الموارد البشرية، يستهدف مديري الموارد البشرية وقادة الأعمال.
النغمة: مهنية ورؤيوية.
الهدف: إظهار الخبرة وجذب العملاء المحتملين.
الطول: متوسط (300-400 كلمة).
يتضمن: مقدمة عن تأثير الذكاء الاصطناعي على التوظيف، 3 نقاط رئيسية (تحسين كفاءة التوظيف، تقليل التحيز، تحسين تجربة المرشح)، وخاتمة مع دعوة للتفاعل (ما هي تجربتك مع الذكاء الاصطناعي في التوظيف؟).
```

### 4.2 كتابة مقالات طويلة

**الأمر**:
```
اكتب مقالاً لـ LinkedIn عن [الموضوع]، يستهدف [الجمهور المهني].
النغمة: [مهنية، تحليلية، تعليمية]
الهدف: [بناء سلطة موضوعية، جذب متابعين]
الطول: طويل (1000-1500 كلمة)
يشمل: مقدمة، عناوين فرعية، نقاط مدعومة بأمثلة، وخاتمة.
```

**مثال عملي**:
```
اكتب مقالاً لـ LinkedIn عن كيفية بناء علامة تجارية شخصية ناجحة في 2026، يستهدف المهنيين الطموحين.
النغمة: تعليمية وعملية.
الهدف: بناء سلطة موضوعية وجذب متابعين جدد.
الطول: طويل (1000-1200 كلمة).
يشمل: مقدمة عن أهمية العلامة الشخصية، 5 استراتيجيات عملية مع أمثلة، نصائح للاستمرار، وخاتمة تحفيزية.
```

### 4.3 كتابة تحديثات وظيفية

**الأمر**:
```
اكتب تحديثاً وظيفياً لـ LinkedIn عن [الإنجاز/الخبرة الجديدة]، بأسلوب احترافي ومتواضع.
```

**مثال عملي**:
```
اكتب تحديثاً وظيفياً لـ LinkedIn عن انضمامي إلى شركة XYZ كمدير تسويق، بأسلوب احترافي ومتواضع.
يتضمن: شكر للفريق السابق، حماس للدور الجديد، ورؤية مختصرة لما أخطط لتحقيقه.
```

### 4.4 كتابة رسائل التواصل المهني

**الأمر**:
```
اكتب رسالة تواصل مهني لـ LinkedIn إلى [الشخص/الجهة] بخصوص [الموضوع]، بأسلوب مهني ولطيف.
```

**مثال عملي**:
```
اكتب رسالة تواصل مهني لـ LinkedIn إلى مدير توظيف في شركة XYZ بخصوص فرصة عمل في قسم التسويق، بأسلوب مهني ولطيف.
يتضمن: تعريف مختصر عن نفسي، إبداء الإعجاب بالشركة، وطلب فرصة لمناقشة الدور.
```

## 5. أوامر فيسبوك (Facebook)

### 5.1 كتابة منشورات للمجموعات والصفحات

**الأمر**:
```
اكتب منشوراً لفيسبوك عن [الموضوع]، يستهدف [الجمهور].
النغمة: [ودودة، إعلامية، تفاعلية]
الهدف: [زيادة التفاعل، بناء مجتمع]
الطول: متوسط (150-250 كلمة)
يتضمن: سؤال للمتابعين أو دعوة للتعليق.
```

**مثال عملي**:
```
اكتب منشوراً لفيسبوك عن نصائح للسفر بأمان، يستهدف المسافرين من العائلات.
النغمة: ودودة ومفيدة.
الهدف: زيادة التفاعل وبناء مجتمع من المسافرين.
الطول: متوسط (200 كلمة).
يتضمن: نصائح عملية، وسؤال للمتابعين عن أفضل وجهة سافروا إليها.
```

### 5.2 إنشاء إعلانات فيسبوك

**الأمر**:
```
اكتب نصاً لإعلان فيسبوك عن [المنتج/الخدمة]، يستهدف [الجمهور].
النغمة: [مقنعة، حماسية]
الهدف: [زيادة المبيعات، جذب العملاء المحتملين]
يتضمن: عنوان جذاب، وصف، ودعوة للاتخاذ إجراء (CTA).
```

**مثال عملي**:
```
اكتب نصاً لإعلان فيسبوك عن دورة تدريبية في التسويق الرقمي، يستهدف رواد الأعمال والمسوقين الجدد.
النغمة: مقنعة وملهمة.
الهدف: زيادة التسجيل في الدورة.
يتضمن: عنوان جذاب ("حوّل شغفك إلى مهنة ناجحة")، وصف للدورة ومميزاتها، ودعوة للتسجيل الآن.
```

### 5.3 إدارة المجموعات

**الأمر**:
```
اكتب رسالة ترحيب لأعضاء جدد في مجموعة فيسبوك عن [الموضوع]، مع شرح لقواعد المجموعة وكيفية المشاركة.
```

**مثال عملي**:
```
اكتب رسالة ترحيب لأعضاء جدد في مجموعة فيسبوك عن ريادة الأعمال، مع شرح لقواعد المجموعة (لا إعلانات، احترام الآخرين) وكيفية المشاركة بفعالية.
النغمة: ودودة ومرحبة.
```

## 6. أوامر تيك توك (TikTok)

### 6.1 كتابة نصوص الفيديوهات

**الأمر**:
```
اكتب سيناريو لفيديو تيك توك عن [الموضوع]، مدته [عدد] ثوانٍ، مع وصف للمشاهد والصوت والنص المعروض على الشاشة.
```

**مثال عملي**:
```
اكتب سيناريو لفيديو تيك توك عن وصفة صحية سريعة، مدته 30 ثانية، مع وصف للمشاهد (تقطيع الخضار، الطهي، التقديم)، والصوت (موسيقى حماسية)، والنص المعروض على الشاشة (المكونات، الخطوات).
```

### 6.2 توليد أفكار لتحديات

**الأمر**:
```
اقترح [عدد] أفكاراً لتحديات (Challenges) على تيك توك عن [الموضوع]، مع شرح لكل فكرة والهاشتاغات المناسبة.
```

**مثال عملي**:
```
اقترح 5 أفكاراً لتحديات على تيك توك عن اللياقة البدنية، مع شرح لكل فكرة (مدة التحدي، القواعد، طريقة المشاركة) والهاشتاغات المناسبة (#FitnessChallenge #Workout).
```

### 6.3 كتابة نصوص تعليقات صوتية

**الأمر**:
```
اكتب نصاً تعليقياً صوتياً (Voiceover) لفيديو تيك توك عن [الموضوع]، بأسلوب حماسي وجذاب.
```

**مثال عملي**:
```
اكتب نصاً تعليقياً صوتياً لفيديو تيك توك عن رحلة يوم في الحياة (Day in the Life) لرائد أعمال، بأسلوب حماسي وجذاب، يتضمن نقاطاً رئيسية عن الروتين اليومي والإنتاجية.
```

## 7. أوامر متعددة المنصات

### 7.1 إعادة توظيف المحتوى عبر المنصات

**الأمر**:
```
خذ المحتوى التالي [نص المحتوى] وأعد كتابته ليناسب [المنصة]، مع الحفاظ على الجوهر مع تعديل النغمة والطول.
```

**مثال عملي**:
```
خذ المقال التالي عن مستقبل الذكاء الاصطناعي وأعد كتابته ليناسب تويتر (خيط تغريدات) وإنستغرام (منشور مع صورة) ولينكد إن (منشور احترافي).
مع الحفاظ على الجوهر مع تعديل النغمة والطول لكل منصة.
```

### 7.2 خطة محتوى متكاملة

**الأمر**:
```
ضع خطة محتوى متكاملة لمدة [فترة زمنية] لمنصات [المنصات] حول [الموضوع]، تشمل عدد المنشورات لكل منصة، الموضوعات الأسبوعية، وأنواع المحتوى.
```

**مثال عملي**:
```
ضع خطة محتوى متكاملة لمدة شهر لمنصات (إنستغرام، تويتر، لينكد إن، تيك توك) حول موضوع الصحة النفسية في مكان العمل، تشمل عدد المنشورات لكل منصة (4 لكل منصة في الأسبوع)، الموضوعات الأسبوعية (الأسبوع الأول: التوتر، الثاني: التواصل، الثالث: التوازن، الرابع: الإيجابية)، وأنواع المحتوى (نصوص، فيديوهات، إنفوجرافيك).
```

### 7.3 تحليل المنافسين

**الأمر**:
```
حلل حسابات [المنافسين] على [المنصة] وقدم تقريراً عن: نوع المحتوى الذي ينشرونه، تكرار النشر، مستوى التفاعل، ونقاط القوة والضعف.
```

**مثال عملي**:
```
حلل حسابات 3 منافسين رئيسيين على إنستغرام في مجال اللياقة البدنية وقدم تقريراً عن: نوع المحتوى الذي ينشرونه (تمارين، تغذية، تحفيز)، تكرار النشر (يومي، أسبوعي)، مستوى التفاعل (متوسط الإعجابات والتعليقات)، ونقاط القوة (جودة الصور، التنوع) والضعف (قلة التفاعل مع المتابعين، تكرار المحتوى).
```

## 8. أوامر للتحليل والتحسين

### 8.1 تحليل أداء المحتوى

**الأمر**:
```
حلل أداء منشوراتي على [المنصة] خلال [الفترة الزمنية]، وقدم توصيات لتحسين الأداء.
البيانات: [إحصائيات التفاعل، عدد المشاهدات، التعليقات، المشاركات]
```

**مثال عملي**:
```
حلل أداء منشوراتي على إنستغرام خلال الشهر الماضي، وقدم توصيات لتحسين الأداء.
البيانات: متوسط الإعجابات (500)، التعليقات (30)، المشاركات (15)، المدى (Reach) (5000).
التوصيات المطلوبة: تحسين أوقات النشر، أنواع المحتوى الأكثر تفاعلاً، اقتراحات للموضوعات القادمة.
```

### 8.2 تحسين الهاشتاغات

**الأمر**:
```
اقترح [عدد] هاشتاغات فعالة لمنشور عن [الموضوع] على [المنصة]، مع تصنيفها إلى: شائعة، متوسطة، ومتخصصة (Niche).
```

**مثال عملي**:
```
اقترح 20 هاشتاغاً فعالاً لمنشور عن السفر إلى إيطاليا على إنستغرام، مع تصنيفها إلى: شائعة (#Travel #Italy)، متوسطة (#ExploreItaly #ItalianFood)، ومتخصصة (#HiddenGemsItaly #ItalianVillages).
```

### 8.3 تحديد أوقات النشر المثلى

**الأمر**:
```
بناءً على تحليل جمهوري على [المنصة]، اقترح أوقات النشر المثلى وأيام الأسبوع الأكثر تفاعلاً.
البيانات: [الموقع الجغرافي للجمهور، أعمارهم، اهتماماتهم]
```

**مثال عملي**:
```
بناءً على تحليل جمهوري على إنستغرام (الجمهور في الشرق الأوسط، أعمار 25-35، مهتمون باللياقة والتغذية)، اقترح أوقات النشر المثلى وأيام الأسبوع الأكثر تفاعلاً.
```

## 10. الخلاصة

أوامر ChatGPT لوسائل التواصل الاجتماعي هي مفتاح تحويل هذه الأداة إلى شريك إبداعي ينتج محتوى جذاب ومناسب لكل منصة. باتباع الهيكل الصحيح، وتحديد الأهداف والجمهور، واستخدام الأوامر المخصصة، يمكنك توفير الوقت وتحسين جودة محتواك.

**الخلاصة في نقاط**:
- **خصص الأوامر لكل منصة**: إنستغرام، تويتر، لينكد إن، فيسبوك، تيك توك.
- **حدد النغمة والهدف**: كل منصة تتطلب نغمة مختلفة (مرحة، مهنية، تعليمية).
- **استخدم أوامر متابعة**: حسّن النتائج تدريجياً.
- **حلل الأداء**: استخدم ChatGPT لتحليل تفاعل جمهورك وتحسين استراتيجيتك.
- **دمج مع ميزات ChatGPT**: الذاكرة، Projects، Canvas لنتائج أفضل.

**توصيتنا**: ابدأ بمنصة واحدة، جرب الأوامر الواردة في هذا الدليل، وعدّلها لتتناسب مع صوت علامتك التجارية. مع الممارسة، ستصبح خبيراً في استخدام ChatGPT لإنتاج محتوى وسائل التواصل الذي يبني مجتمعاً ويزيد التفاعل.
