# استخدام GitHub Copilot في VS Code: الدليل الشامل 2026

في عام 2026، أصبح **GitHub Copilot** واحداً من أكثر الإضافات استخداماً في VS Code. بفضل تكامله السلس مع VS Code، يمكن للمطورين الاستفادة من قوة الذكاء الاصطناعي دون مغادرة بيئة التطوير المفضلة لديهم.

هذا الدليل يقدم لك كل ما تحتاج معرفته عن استخدام GitHub Copilot في VS Code: التثبيت، التهيئة، الميزات الأساسية والمتقدمة، ونصائح لتحقيق أقصى استفادة.

---

## 1. لماذا تستخدم GitHub Copilot في VS Code؟

**VS Code** هو المحرر الأكثر شعبية بين المطورين، وتكامله مع **GitHub Copilot** يجعله أداة لا غنى عنها للبرمجة بالذكاء الاصطناعي.

### 1.1 فوائد التكامل

| الفائدة | الوصف |
|---------|-------|
| **تكامل سلس** | يعمل Copilot مباشرة داخل VS Code بدون إعدادات معقدة |
| **واجهة مألوفة** | جميع ميزات VS Code متاحة مع Copilot |
| **إكمال تلقائي فوري** | اقتراحات أثناء الكتابة مباشرة |
| **دردشة مدمجة** | Copilot Chat داخل المحرر |
| **فهم السياق** | يقرأ الملفات المفتوحة والمشروع بأكمله |
| **تحديثات مستمرة** | تحسينات مستمرة من GitHub و OpenAI |

## 2. تثبيت GitHub Copilot في VS Code

### 2.1 متطلبات التثبيت

| المتطلب | التفاصيل |
|---------|----------|
| **VS Code** | الإصدار 1.70 أو أحدث |
| **حساب GitHub** | حساب نشط |
| **اشتراك Copilot** | مجاني (لمطوري المصادر المفتوحة والطلاب) أو مدفوع |
| **اتصال بالإنترنت** | للتواصل مع خوادم Copilot |

### 2.2 خطوات التثبيت

**الخطوة 1: تثبيت إضافة GitHub Copilot**

1. افتح **VS Code**.
2. اذهب إلى **Extensions** (`Ctrl+Shift+X`).
3. ابحث عن **"GitHub Copilot"**.
4. اضغط على **"Install"**.

**الخطوة 2: تثبيت إضافة GitHub Copilot Chat**

1. في نفس نافذة الإضافات، ابحث عن **"GitHub Copilot Chat"**.
2. اضغط على **"Install"**.

**الخطوة 3: تسجيل الدخول إلى GitHub**

1. بعد التثبيت، ستظهر نافذة تطلب تسجيل الدخول.
2. اضغط على **"Sign in to GitHub"**.
3. اتبع التعليمات لإكمال المصادقة.

**الخطوة 4: تفعيل الاشتراك**

1. بعد تسجيل الدخول، ستظهر نافذة لتفعيل الاشتراك.
2. اختر الخطة المناسبة (مجاني أو مدفوع).
3. أكمل عملية التفعيل.

### 2.3 التحقق من التثبيت

1. افتح ملف كود (مثل `index.js` أو `main.py`).
2. ابدأ بكتابة تعليق، مثلاً:
   ```python
   # دالة لحساب مجموع قائمة الأرقام
   ```
3. إذا ظهر اقتراح رمادي، فهذا يعني أن Copilot يعمل.

## 3. الميزات الأساسية في VS Code

### 3.1 الإكمال التلقائي (Autocomplete)

**الوصف**: أثناء كتابة الكود، يقدم Copilot اقتراحات سطراً بسطر.

**مثال**:
```python
# دالة لجلب البيانات من API
def fetch_data(url):
    # سيظهر اقتراح تلقائي للكود
    import requests
    response = requests.get(url)
    return response.json()
```

**مفاتيح الاختصار**:
- `Tab`: قبول الاقتراح.
- `Esc`: رفض الاقتراح.
- `Ctrl+Arrow` (Windows/Linux) أو `Cmd+Arrow` (macOS): التنقل بين الاقتراحات البديلة.
- `Ctrl+Space` (Windows/Linux) أو `Cmd+Space` (macOS): عرض اقتراحات إضافية.

### 3.2 Copilot Chat

**الوصف**: دردشة مع الذكاء الاصطناعي داخل VS Code.

**فتح Copilot Chat**:
1. من الشريط الجانبي: اضغط على أيقونة **Copilot**.
2. اختصار لوحة المفاتيح: `Ctrl+Shift+I` (Windows/Linux) أو `Cmd+Shift+I` (macOS).

**أمثلة الأسئلة**:
> "كيف يمكنني تحسين أداء هذه الدالة؟"
> "ما هي أفضل ممارسة لكتابة مكون React؟"
> "هل هذا الكود آمن؟"
> "اكتب لي مثالاً لاستخدام مكتبة Axios."

### 3.3 الإكمال المتعدد الأسطر (Multiline Completions)

**الوصف**: اقتراحات تمتد عبر عدة أسطر للدوال المعقدة.

**مثال**:
```javascript
// دالة للاتصال بواجهة برمجة تطبيقات (API) ومعالجة الأخطاء
async function apiCall(url, options = {}) {
    // سيتم توليد الكود الكامل مع معالجة الأخطاء
    try {
        const response = await fetch(url, options);
        if (!response.ok) throw new Error(`HTTP error! status: ${response.status}`);
        const data = await response.json();
        return { success: true, data };
    } catch (error) {
        console.error('API call failed:', error);
        return { success: false, error: error.message };
    }
}
```

### 3.4 الإكمال في التعليقات

**الوصف**: يمكن لـ Copilot توليد كود من التعليقات.

**مثال**:
```python
# أنشئ قائمة تحتوي على الأرقام الزوجية من 1 إلى 20
even_numbers = [x for x in range(1, 21) if x % 2 == 0]
```

### 3.5 الإكمال في ملفات متعددة

يفهم Copilot السياق من الملفات المفتوحة والمشروع بأكمله، مما يحسن جودة الاقتراحات.

**الأمر**:

**النتيجة**: توثيق كامل وجاهز.

**النتيجة**: كود محوّل جاهز.

## 5. تخصيص GitHub Copilot في VS Code

### 5.1 إعدادات Copilot

**فتح الإعدادات**:
1. `Ctrl+,` (Windows/Linux) أو `Cmd+,` (macOS).
2. ابحث عن **"Copilot"**.

**الإعدادات المفيدة**:

| الإعداد | الوصف | القيمة الموصى بها |
|---------|-------|-------------------|
| `editor.inlineSuggest.enabled` | تمكين/تعطيل الاقتراحات | `true` |
| `github.copilot.enable` | تمكين/تعطيل Copilot | `true` |
| `github.copilot.chat.enable` | تمكين/تعطيل Copilot Chat | `true` |
| `github.copilot.advanced` | إعدادات متقدمة | حسب الحاجة |

### 5.2 تعطيل Copilot لملفات معينة

إذا كنت تعمل على ملفات حساسة، يمكنك تعطيل Copilot:

```json
{
  "github.copilot.enable": {
    "*.secret.js": false,
    "*.env": false
  }
}
```

### 5.3 تخصيص اختصارات لوحة المفاتيح

**فتح اختصارات لوحة المفاتيح**:
- `Ctrl+K Ctrl+S` (Windows/Linux) أو `Cmd+K Cmd+S` (macOS).

**اختصارات Copilot المهمة**:

| الاختصار (Windows/Linux) | الاختصار (macOS) | الوظيفة |
|--------------------------|------------------|----------|
| `Ctrl+Shift+I` | `Cmd+Shift+I` | فتح Copilot Chat |
| `Tab` | `Tab` | قبول الاقتراح |
| `Esc` | `Esc` | رفض الاقتراح |
| `Ctrl+Arrow` | `Cmd+Arrow` | التنقل بين الاقتراحات |

## 6. نصائح للاستخدام الفعال في VS Code

### 6.1 اكتب تعليقات واضحة ومفصلة

كلما كانت التعليقات أكثر وضوحاً، كانت الاقتراحات أفضل.

**بدلاً من**:
```python
# مضاعفة الأرقام
def double(x):
    return x * 2
```

**قل**:
```python
# مضاعفة قائمة من الأرقام وإرجاع قائمة جديدة مع العناصر المضاعفة
def double_list(numbers):
    return [n * 2 for n in numbers]
```

### 6.2 استخدم أسماء دوال ومتغيرات دالة

الأسماء الدالة تساعد Copilot على فهم السياق.

### 6.3 افتح ملفات ذات صلة

يفهم Copilot السياق من الملفات المفتوحة، لذا افتح الملفات ذات الصلة بالمهمة.

### 6.4 استخدم اقتراحات بديلة

إذا لم يعجبك الاقتراح الأول، استخدم `Ctrl+Arrow` (Windows/Linux) أو `Cmd+Arrow` (macOS) لعرض اقتراحات بديلة.

### 6.5 تحديث Copilot بانتظام

تأكد من تحديث إضافات Copilot و VS Code للحصول على أحدث الميزات.

## 8. حل المشكلات الشائعة في VS Code

### 8.1 Copilot لا يظهر اقتراحات

**الحل**:
1. تأكد من أنك مسجل الدخول إلى GitHub.
2. تأكد من أن الاشتراك نشط.
3. تحقق من تمكين `editor.inlineSuggest.enabled` في الإعدادات.
4. أعد تحميل VS Code.

### 8.2 Copilot Chat لا يعمل

**الحل**:
1. تأكد من تثبيت إضافة **GitHub Copilot Chat**.
2. تحقق من تمكين `github.copilot.chat.enable`.
3. سجل الخروج وتسجيل الدخول مرة أخرى.

### 8.3 الاقتراحات غير دقيقة

**الحل**:
1. اكتب تعليقات أكثر وضوحاً.
2. افتح ملفات ذات صلة لفهم السياق.
3. استخدم اقتراحات بديلة (`Ctrl+Arrow`).

### 8.4 Copilot بطيء

**الحل**:
1. تأكد من اتصال إنترنت جيد.
2. أغلق الملفات غير الضرورية.
3. حدّث VS Code و Copilot.

> **للمزيد**: راجع [حل مشكلات ChatGPT الشائعة](/posts/chatgpt-common-problems).

## 9. الخلاصة

استخدام GitHub Copilot في VS Code يوفر تجربة برمجة مدعومة بالذكاء الاصطناعي سلسة وقوية. سواء كنت مطوراً مبتدئاً أو محترفاً، يمكن لـ Copilot مساعدتك في كتابة أكواد أسرع وأفضل.

**الخلاصة في نقاط**:
- **التثبيت**: بسيط وسريع من متجر إضافات VS Code.
- **الإكمال التلقائي**: اقتراحات فورية أثناء الكتابة.
- **Copilot Chat**: دردشة داخل المحرر لحل المشكلات.
- **الميزات المتقدمة**: إصلاح الأخطاء، توليد الاختبارات، توثيق الكود.
- **التخصيص**: إعدادات مرنة للتحكم في Copilot.
- **التكامل**: يعمل بسلاسة مع ميزات VS Code الأخرى.

**توصيتنا**: جرب GitHub Copilot في VS Code اليوم مع الفترة التجريبية المجانية. اكتشف كيف يمكن للذكاء الاصطناعي أن يغير طريقة برمجتك.
