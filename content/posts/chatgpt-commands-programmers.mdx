# أفضل أوامر ChatGPT للمبرمجين: دليل سريع 2026

كتابة الأوامر (Prompts) الصحيحة لـ ChatGPT هي الفرق بين نتيجة مفيدة ونتيجة غير مجدية.

هذا الدليل يقدم لك أفضل الأوامر للمبرمجين. استخدمها كما هي، أو عدلها حسب احتياجاتك.

للاستخدام المتقدم والممارسات الشاملة، يمكنك الاطلاع على دليل [كيفية استخدام ChatGPT في البرمجة](/posts/chatgpt-programming-guide) الذي يشرح بالتفصيل كيفية توظيف ChatGPT في سير عملك اليومي.

## فئة: كتابة الكود

### أمر لكتابة دالة

> اكتب دالة بلغة [اللغة] تقوم بـ [الوظيفة].
> المدخلات: [وصف المدخلات]
> المخرجات: [وصف المخرجات]
> متطلبات إضافية: [مثل: معالجة الأخطاء، كفاءة عالية]

**مثال:**

> "اكتب دالة بلغة Python تقوم بتحويل ملف CSV إلى JSON. المدخلات: مسار ملف CSV. المخرجات: كائن JSON. تعامل مع القيم المفقودة."

### أمر لكتابة كلاس

> اكتب كلاس بلغة [اللغة] باسم [الاسم].
> الخصائص: [الخصائص المطلوبة]
> الطرق: [الطرق المطلوبة]
> متطلبات إضافية: [مثل: تطبيق واجهة معينة، استخدام نمط تصميم]

**مثال:**

> "اكتب كلاس بلغة TypeScript باسم User. الخصائص: id، name، email، role. الطرق: updateProfile، getPermissions. استخدم واجهة IUser."

### أمر لكتابة API كامل

> اكتب API بسيطاً باستخدام [الإطار] بلغة [اللغة].
> يحتوي على:
> - نقاط النهاية: [القائمة]
> - التخزين: [نوع التخزين]
> - معالجة الأخطاء: [نعم/لا]
> - المصادقة: [نوع المصادقة]

**مثال:**

> "اكتب API بسيطاً باستخدام Express.js في Node.js. يحتوي على نقاط نهاية CRUD للمنتجات. استخدم MongoDB للتخزين. أضف معالجة للأخطاء."

### أمر لكتابة دالة مع اختبارات

> اكتب دالة بلغة [اللغة] تقوم بـ [الوظيفة].
> ثم اكتب اختبارات لها باستخدام [إطار الاختبار].
> تأكد من تغطية الحالات الطبيعية والحدودية.

## نصائح لكتابة الأوامر الفعالة

### 1. كن محدداً

- **ضعيف:** "ساعدني في البرمجة"
- **جيد:** "اكتب دالة بلغة Python تقوم بحساب المتوسط الحسابي لقائمة الأرقام"

### 2. قدم السياق

- اشرح المشروع ككل
- اذكر اللغات والأطر المستخدمة
- حدد القيود والمتطلبات

### 3. استخدم المحادثات الطويلة

- استمر في نفس المحادثة للحفاظ على السياق
- اطلب تحسينات تدريجية
- راجع النتائج واطلب تعديلات

### 4. اطلب الشرح

- لا تكتفِ بالكود، اطلب شرحاً
- اسأل عن الخيارات التصميمية
- فهم المنطق وراء الكود

### 5. استخدم أمثلة

- قدم أمثلة على المدخلات والمخرجات المطلوبة
- أظهر تنسيق النتيجة التي تريدها

## الأخطاء الشائعة في كتابة الأوامر

### خطأ: "الأمر العام جداً"

**المشكلة:** طلب غامض مثل "اكتب كوداً لموقع".

**الحل:** كن محدداً. اذكر التقنيات، الصفحات، والوظائف المطلوبة.

### خطأ: "عدم تقديم السياق"

**المشكلة:** طلب كود دون شرح المشكلة أو المتطلبات.

**الحل:** اشرح السياق الكامل للمشروع.

### خطأ: "عدم طلب الشرح"

**المشكلة:** استخدام الكود دون فهمه.

**الحل:** اطلب شرحاً لكل جزء من الكود.
