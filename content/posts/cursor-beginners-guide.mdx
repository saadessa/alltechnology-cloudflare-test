# Cursor للمبتدئين: دليل البداية السريع 2026

في عام 2026، أصبحت بيئات التطوير المدعومة بالذكاء الاصطناعي جزءاً أساسياً من سير عمل المطورين. من بين هذه الأدوات، يبرز **Cursor** كواحد من أقوى الخيارات وأكثرها سهولة، خاصة للمبتدئين. يجمع Cursor بين قوة VS Code وميزات الذكاء الاصطناعي المتقدمة في بيئة واحدة متكاملة.

هذا الدليل يقدم لك كل ما تحتاج معرفته لبدء استخدام Cursor كمبتدئ: التثبيت، الاستيراد من VS Code، الميزات الأساسية، والبدء في البرمجة مع الذكاء الاصطناعي.

---

## 1. ما هو Cursor؟

**Cursor** هو بيئة تطوير متكاملة (IDE) قائمة بذاتها مبنية على VS Code، مع ميزات ذكاء اصطناعي متقدمة مدمجة بشكل أساسي. يتيح لك:

- البرمجة مع مساعد ذكي يفهم قاعدة الشيفرة الخاصة بك
- طرح الأسئلة والحصول على شروحات فورية
- توليد أكواد من وصف نصي
- إعادة هيكلة الكود تلقائياً
- فهم المشاريع الكبيرة بسرعة

### 1.1 مميزات Cursor للمبتدئين

| الميزة | الفائدة للمبتدئين |
|--------|-------------------|
| **محادثة مع الشيفرة** | اسأل عن أي جزء من الكود وافهمه بسرعة |
| **توليد الكود** | اكتب وصفاً باللغة الطبيعية واحصل على كود جاهز |
| **الإكمال التلقائي** | اقتراحات أثناء الكتابة توفر الوقت |
| **شرح الكود** | فهم الكود المعقد بسهولة |
| **واجهة مألوفة** | تشبه VS Code، سهلة التعلم |
| **استيراد الإعدادات** | انقل إعدادات VS Code بنقرة واحدة |

## 2. تثبيت Cursor

### 2.1 تحميل Cursor

1. اذهب إلى [الموقع الرسمي لـ Cursor](https://cursor.com).
2. اضغط على **"Download"**.
3. اختر الإصدار المناسب لنظام التشغيل الخاص بك:
   - **Windows**: ملف `.exe`
   - **macOS**: ملف `.dmg`
   - **Linux**: ملف `.AppImage` أو `.deb`

### 2.2 تثبيت Cursor

**على Windows**:
1. شغّل ملف التثبيت `.exe`.
2. اتبع التعليمات على الشاشة.
3. بعد التثبيت، شغّل Cursor من قائمة "ابدأ".

**على macOS**:
1. افتح ملف `.dmg`.
2. اسحب Cursor إلى مجلد **"التطبيقات"**.
3. شغّل Cursor من مجلد التطبيقات.

**على Linux**:
1. شغّل ملف `.AppImage` أو ثبّت ملف `.deb`.
2. شغّل Cursor من قائمة التطبيقات.

### 2.3 التثبيت عبر سطر الأوامر (اختياري)

**macOS (via Homebrew)**:
```bash
brew install --cask cursor
```

## 3. استيراد إعدادات VS Code (للمستخدمين السابقين)

إذا كنت تستخدم VS Code حالياً، يمكنك استيراد جميع إعداداتك بسهولة:

1. عند فتح Cursor لأول مرة، ستظهر نافذة ترحيب.
2. اختر **"Import from VS Code"**.
3. اختر الإعدادات التي تريد استيرادها:
   - الإضافات (Extensions)
   - الإعدادات (Settings)
   - اختصارات لوحة المفاتيح (Keybindings)
   - مقتطفات الكود (Snippets)
4. اضغط على **"Import"**.
5. انتظر حتى يكتمل الاستيراد.

**ملاحظة**: إذا لم تظهر نافذة الاستيراد، يمكنك استيراد الإعدادات يدوياً من خلال:
- **File** > **Import from VS Code**

## 4. واجهة Cursor الأساسية

### 4.1 نظرة عامة على الواجهة

واجهة Cursor تشبه إلى حد كبير VS Code، مما يجعل التعلم سهلاً:

| العنصر | الوصف |
|--------|-------|
| **الشريط الجانبي (Sidebar)** | يحتوي على مستكشف الملفات، البحث، Git، والإضافات |
| **شريط الحالة (Status Bar)** | يعرض معلومات عن المشروع والحالة |
| **منطقة التحرير (Editor)** | مكان كتابة وتحرير الكود |
| **الطرفية (Terminal)** | سطر الأوامر المدمج |
| **نافذة المحادثة (Chat)** | التفاعل مع الذكاء الاصطناعي |

### 4.2 الفروقات عن VS Code

| العنصر | VS Code | Cursor |
|--------|---------|--------|
| **أيقونة المحادثة** | غير موجودة | أيقونة دردشة في الشريط الجانبي |
| **الإكمال التلقائي** | يحتاج إضافات | مدمج ومتقدم |
| **اقتراحات الذكاء الاصطناعي** | من خلال إضافات | مدمجة بشكل أساسي |

## 5. الميزات الأساسية للمبتدئين

### 5.1 المحادثة مع الذكاء الاصطناعي (Chat)

يمكنك التحدث مع الذكاء الاصطناعي مباشرة داخل المحرر:

**فتح نافذة المحادثة**:
- `Cmd+Shift+L` (macOS) أو `Ctrl+Shift+L` (Windows)

**أمثلة على الأسئلة**:
> "ما هي هذه الدالة؟ اشرحها لي."
> "كيف يمكنني تحسين هذا الكود؟"
> "اكتب دالة لحساب متوسط الأرقام."

**المحادثة المضمّنة (Inline Chat)**:
- حدد جزءاً من الكود.
- اضغط `Cmd+L` (macOS) أو `Ctrl+L` (Windows).
- اكتب سؤالك عن الكود المحدد.

### 5.2 توليد الكود (Code Generation)

يمكنك توليد كود من وصف نصي:

**الأمر**:
> "اكتب دالة Python لجمع الأرقام الزوجية من 1 إلى 100."

**النتيجة**: كود جاهز مع شرح.

**الأمر**:
> "أنشئ مكون React لعرض قائمة المنتجات."

**النتيجة**: مكون React كامل.

### 5.3 الإكمال التلقائي (Autocomplete)

أثناء كتابة الكود، يقدم Cursor اقتراحات تلقائية:

- **اقتراح سطر كامل**: أثناء الكتابة، يظهر اقتراح رمادي.
- **قبول الاقتراح**: اضغط `Tab`.
- **رفض الاقتراح**: استمر في الكتابة.

**نصيحة**: الإكمال التلقائي يعمل بشكل أفضل مع المشاريع الكبيرة التي يفهم سياقها.

### 5.4 شرح الكود (Explain Code)

لفهم كود معين:

1. حدد الكود الذي تريد شرحه.
2. اضغط `Cmd+L` (macOS) أو `Ctrl+L` (Windows).
3. اكتب: "اشرح هذا الكود."

**النتيجة**: شرح مبسط ومفصل للكود المحدد.

### 5.5 إعادة الهيكلة (Refactoring)

لتحسين الكود:

1. حدد الكود.
2. اضغط `Cmd+L` (macOS) أو `Ctrl+L` (Windows).
3. اكتب: "أعد هيكلة هذا الكود ليكون أكثر كفاءة."

**النتيجة**: كود محسّن مع شرح التغييرات.

## 7. العمل مع المشاريع

### 7.1 فتح مشروع موجود

1. في Cursor، اختر **File** > **Open Folder**.
2. اختر مجلد المشروع.
3. سيتم فتح المشروع في نافذة Cursor.

### 7.2 إنشاء مشروع جديد

1. اختر **File** > **New Folder**.
2. أنشئ مجلداً جديداً وافتحه.
3. ابدأ بإنشاء ملفات جديدة.

### 7.3 إدارة الملفات

- **إنشاء ملف جديد**: `Cmd+N` (macOS) أو `Ctrl+N` (Windows).
- **حفظ الملف**: `Cmd+S` (macOS) أو `Ctrl+S` (Windows).
- **فتح ملف**: `Cmd+O` (macOS) أو `Ctrl+O` (Windows).
- **البحث في الملفات**: `Cmd+Shift+F` (macOS) أو `Ctrl+Shift+F` (Windows).

## 8. دمج Git في Cursor

Cursor يدعم Git بشكل مدمج:

- **رؤية التغييرات**: من خلال الشريط الجانبي (أيقونة Git).
- **إضافة التغييرات**: اضغط على `+` بجانب الملفات المعدلة.
- **كتابة رسالة commit**: اكتب الرسالة واضغط `Cmd+Enter`.
- **دفع التغييرات**: اضغط على أيقونة السحابة.

**مفتاح الاختصار**: `Cmd+Shift+G` (macOS) أو `Ctrl+Shift+G` (Windows) لفتح Git.

## 9. نصائح للمبتدئين

### 9.1 ابدأ بمشروع صغير

لا تبدأ بمشروع كبير ومعقد. ابدأ بمشروع صغير مثل:
- تطبيق "مرحباً بالعالم" بلغة Python.
- صفحة ويب بسيطة بـ HTML/CSS.
- تطبيق صغير بلغة JavaScript.

### 9.2 استخدم المحادثة للتعلم

بدلاً من البحث في Google، اسأل Cursor مباشرة:
> "كيف يمكنني عمل حلقة for في Python؟"
> "ما هي أفضل ممارسة لتنظيم ملفات React؟"

### 9.3 اطلب شرح الكود

عندما ترى كوداً لا تفهمه، حدده واسأل:
> "اشرح هذا الكود خطوة بخطوة."

### 9.4 استخدم الإكمال التلقائي بحكمة

لا تقبل جميع الاقتراحات بشكل أعمى. اقرأ الاقتراح، وتأكد من أنه مناسب قبل قبوله.

### 9.5 جرّب توليد الكود

اكتب وصفاً لما تريد، ودع Cursor يولد الكود. ثم اقرأه وافهمه.

### 9.6 استفد من الذاكرة

Cursor يتذكر محادثاتك السابقة، مما يجعله أكثر تخصيصاً وفائدة بمرور الوقت.

## 10. المشكلات الشائعة للمبتدئين وحلولها

### 10.1 لا يعمل الإكمال التلقائي

**الحل**:
1. تأكد من أنك في ملف كود (لغة برمجة مدعومة).
2. ابدأ بكتابة كود بسيط، سيظهر الاقتراح.
3. إذا استمرت المشكلة، أعد تشغيل Cursor.

### 10.2 المحادثة لا تستجيب

**الحل**:
1. تأكد من اتصالك بالإنترنت.
2. إذا كنت تستخدم النسخة المجانية، تأكد من أنك لم تتجاوز الحد اليومي.
3. أعد تشغيل Cursor.

### 10.3 Cursor بطيء

**الحل**:
1. أغلق المشاريع والملفات غير الضرورية.
2. تأكد من أن جهازك يلبي متطلبات النظام.
3. إذا استمرت المشكلة، جرب استخدام Cursor مع مشروع أصغر.

## 11. الخلاصة

Cursor هو أداة قوية ومبتكرة تجعل البرمجة أسهل وأسرع، خاصة للمبتدئين. بفضل ميزات الذكاء الاصطناعي المدمجة، يمكنك تعلم البرمجة وفهم الأكواد وتوليدها بسهولة.

**الخلاصة في نقاط**:
- **تثبيت سهل**: حمّل من الموقع الرسمي وثبّت في دقائق.
- **استيراد الإعدادات**: انقل إعدادات VS Code بسهولة.
- **ميزات AI**: محادثة، توليد كود، إكمال تلقائي، شرح كود، إعادة هيكلة.
- **واجهة مألوفة**: تشبه VS Code، سهلة التعلم.
- **مجتمع ودعم**: موارد متزايدة ومجتمع نشط.

**توصيتنا**: إذا كنت مبتدئاً في البرمجة أو تبحث عن بيئة تطوير مدعومة بالذكاء الاصطناعي، جرب Cursor اليوم. ابدأ بمشروع صغير، استخدم المحادثة للتعلم، واستمتع بتجربة برمجة سلسة وذكية.
